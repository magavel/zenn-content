# AMIの共有方法
1. EC2コンソールからイメージメニューにあるAMI画面を開く
2. 対象のAMIにチェックを入れた状態で「アクション」をクリックする
3. 「イメージパーミッションの変更」を選択する
4. 共有したい別のAWSアカウントIDを追加する
